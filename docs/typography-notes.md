# 佛经刻本排版 · 笔记与预制格式库

本文件三部分：**① 排版规矩笔记**（依据）、**② 预制格式库**（可复用版式模板）、
**③ 应用机制与分期**。总原则：**忠实底本、不代施句读**；纯视觉的预制格式最安全，
凡「给原文加格式」（抬头、偈颂另排等）一律做成**可选开关、默认关**。

---

## 一、排版规矩笔记（依据）

### 装帧
卷轴装（唐）→ **经折装**（佛经诵读主流，连续折叠）→ 线装/包背装。
🚩 **经折装无中缝版心/鱼尾**（那是线装特征）；折缝在折页处。我们现用的「左右版框鱼尾栏」
属线装式样，作**风格化装饰**保留。

### 版框 / 天地 / 行款
- 四周双边（佛经常见）；**天头宽、地脚窄**。
- **界行（乌丝栏）**细竖线分行。
- 折装每面固定行数（常 5–6）、每行固定字数（常 14/15/17/20，逐藏不同）。

### 卷首卷末程式
- 卷首**经题顶格** → 次行**译人题**（低格）→ **品题**分行。
- 卷末**尾题**（再出经名卷次）+ **音義** + **刊记**（缘起/施主/刻工/年月）。
- 明代佛经：卷首**说法图**扉画、卷末**韦陀护法图**。

### 字体 / 朱墨 / 敬避
- 唐宋写经楷 → 明宋体方字。
- **传统正文多白文**；句读圈点系评点/后加 → 我们的朱色句读=现代辅助。
- **敬避抬头（平阙）**：遇「佛/世尊/皇帝/御/天/祖」空格或换行抬高（序、题记、御制最讲究）。

### 御制序（规格最高）
位置最前 · 逢尊必抬头 · **朱色或泥金套印以别尊** · 云龙版框扉饰 · 钤御宝 · 多色套版须套准。

### 其它
千字文帙号编函；刻工署名；大字本/小字本；陀罗尼咒音译另标；天台华严科判。

## 三、应用机制与分期

### 「版式」设置面板（统一入口）
在工具栏加「版式」按钮，集中管理**预制格式的开关**：
- 主题皮肤（宣纸/磁青泥金/木刻）单选；
- 视觉开关（折痕、天地留白、乌丝栏强弱）多选；
- 内容格式开关（偈颂另排、敬空抬头、書衣題簽、扉画）多选，默认关；
- 字体、字数（已有）。
- 存 `localStorage.cbeta_style = {...}`，随经文加载应用。

### 御制序等「专页/专格」的判定
- 优先在 **prep 阶段**给 JSON 加标记（如 `special:'yushu'`、段落 `verse` 区间），渲染端只按标记应用——比渲染端正则更稳、可控。
- 御制识别：题名/首行含「御製」；抬头字表（佛/世尊/如來/皇帝/朕/聖/御/天/祖…）。

### 分期
1. **纯视觉皮肤批**（不碰数据，最快最稳）：天头地脚 + 折痕 + 乌丝栏开关 + 「版式」面板骨架。
2. **書衣·題簽 动态叶**（卷首封面，经名题签）+ 磁青泥金主题。
3. **偈颂另排**（prep 加 verse 标 → 渲染）；音義分区。
4. **御製序 专页**（抬头 + 朱/泥金 + 云龙框）+ 敬空抬头开关。
5. **扉画/护法 线描叶**（可选装饰）。

### 忠实白文红线
- A/C/D 类多为纯视觉或既有结构 → 安全。
- B 类里 **偈颂另排、敬空抬头** 是「给原文加格式」→ 一律**开关、默认关**，且**不改经文数据本身**，只在渲染层呈现。
